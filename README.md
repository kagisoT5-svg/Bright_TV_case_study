This project analyzes transactional data from Bright Coffee Shop to support the CEO’s goal of increasing revenue and improving product performance. The focus is on identifying key sales drivers and providing actionable insights.

Obejectives
-Identify top revenue-generating products
-Determine peak sales times
-Analyze sales trends across products and time
-Provide recommendations to improve performance

Tools Used
-Excel – Data cleaning & exploration
-Databricks – SQL analysis
-Miro – Workflow planning
-Canva – Presentation design
